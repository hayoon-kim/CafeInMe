# Cafe In Me Project

## Description

카페 창업이 어마어마하게 늘어나고 있는 요즘, 공급이 많은 만큼 수요가 현명하게 이루어질 필요가 있어 기획하게 되었습니다.
소비자가 원하는 조건을 직접 선택하여 카페를 찾고 그 카페에 대한 정보를 얻을 수 있는 웹 서비스입니다.


## Environment

JAVA 기반으로 제작되었고 Visual StudioCode로 프론트엔드를 제작하였습니다. 
저는 프론트엔드 전반과 전체 기획을 맡았습니다. 


## Prerequisite

DB가 있어야만 동작하는 웹 사이트입니다.


## Files

이 항목은 내가 추가한 것이다. 중요한 코드 파일들 몇 개를 대상으로 해당 파일이 어떠한 역할을 하는 파일인지를 간단히 설명해주면 전반적인 맥락을 파악하기에 좋을 것 같아 추가하였다.
